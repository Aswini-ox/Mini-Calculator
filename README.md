# Mini-Calculator
🧮 Scientific Calculator (Python + Tkinter)  A modern and attractive Scientific Calculator built using Python and Tkinter. This calculator supports basic arithmetic operations as well as scientific functions like square root, trigonometry, logarithms, and π.
Scientific Calculator (Python + Tkinter)

🌟 Features

➕➖✖️➗ Perform basic operations (Add, Subtract, Multiply, Divide).

√ Square root function.

🔢 Scientific Functions:

sin, cos, tan (in degrees).

log (base 10).

π constant.

🖥️ Attractive GUI with a modern dark theme.

🧮 Easy-to-use button layout.

❌ Error handling for invalid inputs.

🛠️ Tech Stack

Python 3

Tkinter (GUI library)

Math module (for scientific calculations)

📂 Project Structure
├── scientific_calculator.py   # Main calculator script
